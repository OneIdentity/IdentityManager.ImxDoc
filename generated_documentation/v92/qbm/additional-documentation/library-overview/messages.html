IMX Web - QBM Library

Messages and Translation

This section describes components that are used to display messages in different languages to the user. There are the following components and services available.

1. Messages

1.1. User Messages

User messages are displayed using the UserMessageComponent, which contains an Elemental UI alert component.

1.2. Message Dialog

The message dialog is defined in the MessageDialogComponent.
It can display the same information, the normal UserMessageComponent is showing, but it uses an Angular Material Dialog instead.

MessageDialogComponent is part of the QbmModule

1.3. Confirmation Dialog

The confirmation dialog is opened using the ConfirmationService.
It allows the user to acknowledge the information it is presented.

1.4. Snackbar

A possible way to display a small notification is by using an Angular Snackbar component. To open a pre defined snackbar, you can use the SnackBarService. It is part of the QbmModule as well.

1.5. Logging

There is a special logger, that can be used, to send messages to the command line.

The code is defined in the ClassLoggerService

1.6 Error Handling

Error handling is handled by our own ErrorHandler.

It is defined in the GlobalErrorHandler

2.Translation

Our libraries are using translatable keys, identifiable by the prefix #LDS#. The services need to translate these keys into localized text at runtime are described below.

2.1. Translation

This uses the ImxTranslationProviderService, to initialize the translation information. You can use this service for translation purposes as well, but it is recommended to use the TranslateService from @ngx-translate, which is used in this service anyway.

2.2. LDS Replace

The LdsReplaceModule contains a pipe, which can be used to replace placeholder inside LDS key translations. It is defined in the LdsReplacePipe class.

2.3. Parameterized Text

The ParameterizedTextModule provides components and services to display a parameterized text, that emphasizes it's parameters. The UI is defined in the ParameterizedTextComponent.

2.4. Translation Editor

The TranslationEditorComponent is declared in the QbmModule. It can be used to add translations to an LDS key.
