Other Reusable Components and Services

In this section you can find components that are intended to be reused in other libraries and applications, and not covered by the other sections

1. Bulk Editor

The BulkPropertyEditorComponent is used to edit the same list properties for multiple objects. Each BulkItemComponent has it's own list of CdrEditorComponents

2. Busy Indicator

The BusyIndicatorComponent can be used to display a loading spinner and a predefined text. A good solution is to use it together with a BusyService, that indicates if something has started / ended loading.

3. Charts

In our web, charts are rendered using the ElementalUiChartComponent. There are some configurations defined by related classes

  • LineChartOptions that defines a line chart definition
  • SeriesInformation, that defines one series in a line chart
  • XAxisInformation and YAxisInformation, that defines the information for the line axis

There is also a ChartTileComponent, that can be used to render a chart as part of a tile.

4. Custom Theme

The CustomThemeService can be used to load all defined themes. It is also used to add the themes to the <head> part of the page.

5. Dynamic Tabs

Some tabs in side sheets have to be added dynamically because they are part of a dynamic module. The item is defined by a new TabItem and can be added to a tab control using a DynamicTabDataProviderDirective.

6. Extensions

Sometimes it is necessary to add a component, that is not defined as part of a static module, but is part of a dynamic one. To display this component, a ExtComponent can be used. It is part of the ExtModule, together with the needed service and a directive.

7. File Selector

If the user should be able to upload files, there is a FileSelectorService for that.

8. Hyper View

A hyper view is a linked graph that visualizes the relationships of an object to other objects. The HyperviewComponent and its parts are all part of the HyperviewModule.

9. Masthead

In the masthead, information like the company logo, a help icon or the user menu can be displayed. It is defined inside the MastHeadComponent, that is part of the 'MastHeadModule'.

The MastHeadComponent is also responsible for the menu line underneath it. The menu is rendered using an ElementalUiTopNavigationComponent. The menu items can be defined by using the MenuService.

10. Object History

The ObjectHistoryComponent can be used to show the course of an object. It can be display as a table or by using a TimelineComponent.

11. Side Navigation

The SideNavigationViewComponent is used to display a vertical menu, that can be part of a page or an other component. It is collapsible for smaller screens.

Another representation of a side navigation is the SidenavTreeComponent which uses a tree view inside a collapsible panel.

The SideNavigationViewComponent is part of the SideNavigationViewModule while the SidenavTreeComponent is part of the SidenavTreeModule.

12. SQL Wizard

The SqlWizardComponent can be used to filter the result of a larger amount of entries. It generates a filter expression, that can be used as part of a normal API call. The SQL wizard is defined in the SqlWizardModule together this the sub components, that are used to build the filter.

To use a SqlWizardComponent as part of the table filter in a custom project, the abstract class SqlWizardApiService has to be extended and added as a provider.
