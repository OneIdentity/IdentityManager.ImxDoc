Other Reuseable Components and Services

In this section you can find components, that can be reused in other libraries and applications. These components do not fit into the other sections, but might be important as well. They are sorted in alphabetical order and not on importance.

PS: This list is not complete, but it contains the most important ones.

1. Data Explorer \ My Responsibilities

The DataExplorerViewComponent and the MyResponsibilitiesViewComponent are used to display the main component for the _Data administration > Data Explorer and the Responsibilities > My Responsibilities pages. It shows a menu item for each type, that is registered in the DataExplorerRegistryService or the MyResponsibilitiesRegistryService respectively.

2. Delegation

The DelegationComponent can be used to delegate responsibilities to another identity. The component uses ad MatStepper to navigate through the steps.

3. Related Applications

The RelatedApplicationsComponent can be added to the navigation of the portal. It adds links to other sites to the navigation. These links are configured in the database.

4. Risk Index

The RiskConfigComponent displays a table containing all available risk index functions. They can be edited by using the RiskConfigSidesheetComponent.

5. Source Detective

The SourceDetectiveComponent shows the assignment analysis of an object. It contains a tree, with branches for every assignment.

6. Statistics

The StatisticsModule contains all components, that are needed to show statistics for different part of the system. The entry point is the StatisticsHomePageComponent. It contains a tree with its leaves representing single sections of the portal.

Every leaf is equipped with a list of statistic, the user can look into. Clicking on a statistic, opens a ChartsSidesheetComponent or a HeatmapSidesheetComponent, depending on the statistic.

7. Terms of Use

The terms of use for a product can be accepted using the TermsOfUseListComponent. It is possible, to use 2 Factor Authentication, if that is configured correctly by the system.

8. User Process

The UserProcessComponent displays a list of all the processes, that are associated with the currently logged in user. Each process is marked according to his current state.
